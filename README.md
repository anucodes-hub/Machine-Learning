# Machine-Learning Algorithms

This repository contains implementations of fundamental Machine Learning algorithms using Python and Google Colab notebooks.
It focuses on building intuition through hands-on experimentation with datasets, visualization, and model evaluation.

---

## 📂 Repository Contents

### 📘 Core Algorithms

* **Decision Tree.ipynb**
  Implementation of Decision Tree classification.
  Covers training, prediction, and visualization of decision boundaries.

* **Gaussian Naive Bayes Classification.ipynb**
  Implements the Gaussian Naive Bayes algorithm for classification problems.
  Demonstrates probabilistic learning and prediction.

* **K-means Clustering.ipynb**
  Unsupervised learning using K-Means clustering.
  Includes centroid initialization, clustering process, and visualization.

* **Linear Regression Problem.ipynb**
  Solves a regression problem step-by-step using Linear Regression.
  Focuses on prediction and performance evaluation.

* **linear_regression.ipynb**
  Another implementation of Linear Regression (possibly from scratch or variation).
  Helps compare approaches and deepen understanding.

---

### 📊 Data Handling & Visualization

* **Reading_dataset.ipynb**
  Demonstrates how to import and inspect datasets using Pandas.

* **Reading_display_dataset_plots.ipynb**
  Extends dataset reading with visualization.
  Includes plots and graphs to understand data distribution.


2. Open notebooks using:

   * Jupyter Notebook locally

3. Run each notebook step-by-step to understand the workflow.
